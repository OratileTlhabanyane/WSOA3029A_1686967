Rationale

I like a very simplistic website. The less the better. So i chose a colour scheme that was calm but quite bold. The API I initially wanted to use was about looking at the effects of of mental health on Gender-Based Violence in South Africa but I wasn't able to find an API that would have helped me illustrate these statistics. So i chose a colour scheme colours that were relaxing and calm, and a simplisic user interface because GBV is a very sensitive topis so I didn't want my user to feel overwhelmed by the data they are consuming and also be frustrated by the colours or complex interface when interacting with the website.
